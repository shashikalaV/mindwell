# 🧠 Mental Wellness Tracker

A simple web application designed to help users track their mental wellness through mood logging, journaling, and breathing exercises.

## 🚀 Features

* 😊 Mood Tracking
* 📓 Daily Journaling
* 🌬️ Breathing Exercise Sessions
* 👤 User Profile with Editable Avatar
* 📊 Activity Statistics Dashboard
* 💾 Data Persistence using LocalStorage

## 🛠️ Tech Stack

* React.js
* JavaScript
* HTML5
* CSS3
* LocalStorage (for data persistence)

## 📷 Application Modules

* Login & Signup Authentication
* Mood Tracker
* Journal Entries
* Breathing Exercise Tracker
* User Profile Management
* Activity Statistics

## 🎯 Purpose

This project was developed as part of an internship assignment to demonstrate frontend development skills using React and browser storage.

## ▶️ How to Run

1. Clone the repository
2. Install dependencies
3. Run the application

```bash
npm install
npm start
```
